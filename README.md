# SSUBOOK
Used book transaction application  

## 기획 의도

대학 서적을 전문적으로 거래하는 Application 제작  

## 작품 설명

대학생들이 중고 대학 서적을 거래할 수 있는 플랫폼을 제공한다.  

## 차별화 전략 (vs당근마켓)

당근마켓과 달리, 책 이외의 물품이 없다. 
뿐만 아니라, 대학서적 거래 시장이 없기 때문에 확실한 타겟층 확보 가능
